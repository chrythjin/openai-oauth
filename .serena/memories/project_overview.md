openai-oauth is a TypeScript monorepo on Windows that provides OpenAI-compatible local proxy access via ChatGPT/Codex OAuth tokens. Main packages: packages/openai-oauth (CLI/proxy), packages/openai-oauth-core (auth, transport, SSE), packages/openai-oauth-provider (Vercel AI SDK provider), packages/openai-oauth-mcp (MCP integration). Workspace uses Bun and Turbo. Key runtime entrypoint for proxy is packages/openai-oauth/dist/cli.js after build, typically binding to 127.0.0.1:10531.
